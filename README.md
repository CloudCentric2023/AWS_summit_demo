# devpet :feet:
**AI 開発が「AIペットと過ごす時間」に変わるTUI 育成ツール。**

- AI 開発時代の脳疲労、応答待ちの退屈 — ターミナルにいるペットがゆるく話し相手になる
- AIとの開発活動/ 話しかけるボタンでレベルアップ。一定レベルで進化、24 時間放置で死亡
- 「開発してるフリして実はペットとだべってるだけ」になる、人をダメにするクリッカー体験

## 主な機能
- **ペット育成** — AAキャラクターのペットをターミナル上に常駐、育成
- **AI応答** — Amazon Bedrock（Claude Haiku）によるパラメータ連動のゆるい会話
- **進化システム** — たまご → こども → おとな の3段階進化（パラメータ傾向で分岐）
- **4つのパラメータ** — かしこさ / きずな / きぶん / たいりょく(HP) が相互作用
- **死亡・復活** — 24時間放置で力尽きる。新しいたまごで再スタート
- **クラウド同期** — Cognito登録でセーブデータをクラウドに同期（オプション）
- **オフライン対応** — ローカルだけでも育成可能
- **AIツール連携** — Kiro等のhook機能でプロンプト送信を検知してEXP獲得
