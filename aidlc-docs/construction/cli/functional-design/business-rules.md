# cli — ビジネスルール定義

## 1. 話しかけルール

### BR-CLI-01: クールダウンなし
- 「話しかける」にクールダウンは設けない
- 連打可能だが、HP が微減するためバランスが取れる
- 話しかけ1回あたり HP -1〜-3（ランダム）

### BR-CLI-02: 話しかけパラメータ変動

| パラメータ | 変動量 | 備考 |
|-----------|--------|------|
| EXP | +100 | 固定値 |
| きずな | +3〜+5 | ランダム。最も上がるパラメータ |
| かしこさ | +1〜+2 | ランダム。少量UP |
| きぶん | -2〜+3 | ランダム。話しかけで気分が変わる |
| HP | -1〜-3 | ランダム。連打ペナルティ |

### BR-CLI-03: AIツール連携パラメータ変動

| パラメータ | 変動量 | 備考 |
|-----------|--------|------|
| EXP | +10 | 固定値 |
| かしこさ | +1 | 固定値。AIツール使用で賢くなる |

## 2. 時間経過ルール

### BR-CLI-04: パラメータ自然変動（放置時）

| パラメータ | 変動速度 | 24時間放置時の変動 |
|-----------|---------|-----------------|
| HP | -3/時間 | -72 |
| きずな | -2/時間 | -48 |
| かしこさ | -0.5/時間 | -12 |
| きぶん | -1/時間 | -24 |

### BR-CLI-05: HP自然回復（TUI起動中）
- TUI起動中、HPは +0.01/秒 で自然回復（≒ +36/時間）
- ただし上限100を超えない
- 放置中（TUI未起動）はHP回復なし

### BR-CLI-06: きぶん時間帯変動（TUI起動中）
- 6:00〜12:00: きぶん微増傾向（+0.005/秒）
- 12:00〜18:00: きぶん安定（変動なし）
- 18:00〜24:00: きぶん微減傾向（-0.003/秒）
- 0:00〜6:00: きぶん低下（-0.008/秒）

### BR-CLI-07: HP→きぶん連動
- HP ≤ 30 の場合、きぶんの自然低下速度が2倍

## 3. 死亡ルール

### BR-CLI-08: 24時間放置死亡
- TUI起動時に `現在時刻 - LastInteractionAt ≥ 24時間` なら即死亡
- HP を 0 に設定
- CauseOfDeath = "neglect"

### BR-CLI-09: HP枯渇死亡
- Tick処理またはTalk処理でHP=0になった場合に死亡
- CauseOfDeath = "exhaustion"

### BR-CLI-10: 死亡演出
- 墓石AAを表示
- 「…お留守の間に、力尽きてしまいました」等のメッセージ
- 「新しいたまごで再スタートしますか？ [y/n]」の選択肢

## 4. 描画ルール

### BR-CLI-11: リアルタイム更新
- 1秒ごとにパラメータ表示を更新
- bubbletea の Tick メッセージで実装

### BR-CLI-12: 最小ターミナルサイズ

**提案: 80x24（標準最小サイズ）**

レイアウト配分（24行）:
```
行 1     : タイトルバー（devpet + ペット名）
行 2     : 区切り線
行 3-10  : ペットAA表示エリア（8行）
行 11-13 : 吹き出し（3行）
行 14    : 区切り線
行 15-16 : ステータス（LV, EXP, Stage）
行 17-20 : パラメータバー（HP, かしこさ, きずな, きぶん）
行 21    : 区切り線
行 22    : キーバインド表示
行 23-24 : メッセージ/通知エリア
```

80列あれば、パラメータバーのプログレスバー表示やAA表示に十分な幅がある。

### BR-CLI-13: ターミナルサイズ不足時
- 最小サイズ未満の場合、「ターミナルを広げてください」メッセージを表示
- リサイズを検知して自動的にレイアウトを再計算

## 5. カラースキームルール

### BR-CLI-14: カスタムテーマシステム
- テーマは `Theme` 構造体で定義
- 進化タイプごとに異なるカラーパレットを設定可能
- デフォルトテーマを提供し、将来的にキャラごとのテーマ拡張に対応

**デフォルトカラーパレット:**

| 要素 | カラー | 用途 |
|------|--------|------|
| Primary | `#7C3AED` (紫) | タイトル、強調 |
| Secondary | `#10B981` (緑) | ステータス、正常値 |
| Accent | `#F59E0B` (黄) | 警告、注意 |
| Danger | `#EF4444` (赤) | HP低下、死亡 |
| Muted | `#6B7280` (灰) | 非アクティブ、補助テキスト |
| Background | ターミナルデフォルト | 背景はユーザー設定に従う |
| Text | ターミナルデフォルト | テキストはユーザー設定に従う |
