## ユニットごとの機能要件カバレッジ

### shared
- **主担当**: FR-03（パラメータの型定義・バリデーション）
- **関連**: FR-01〜FR-07, FR-09（Pet構造体、API型定義、定数）

### cli
- **主担当**: FR-01（ペット表示）、FR-02（話しかけUI）、FR-03（パラメータ管理）、FR-04（進化演出）、FR-05（レベルアップ演出）、FR-06（死亡・復活画面）、FR-07（ローカル保存・クラウド同期呼び出し）、FR-11（AIツール連携hook）
- **関連**: FR-08（認証クライアント）

### cdk
- **主担当**: FR-10（Webホスティング）
- **関連**: FR-08（Cognito設定）、全バックエンドリソースのプロビジョニング

### backend
- **主担当**: FR-02（AI応答API）、FR-07（クラウドセーブ/ロードAPI）、FR-08（認証API）、FR-09（AI応答生成ロジック）
- **関連**: FR-03（パラメータ変動計算の一部）

### web
- **主担当**: FR-10（紹介Webサイト）
- **関連**: FR-08（サインアップ/サインインUI、デバイス認証承認）

## 非機能要件のユニット適用

| 非機能要件 | 適用ユニット |
|-----------|-------------|
| **NFR-01**: パフォーマンス | cli（TUI描画60fps）、backend（API応答5秒以内） |
| **NFR-02**: 可用性 | cli（オフライン動作）、backend（エラーハンドリング） |
| **NFR-03**: セキュリティ | 全ユニット（SECURITY-01〜15） |
| **NFR-04**: 拡張性 | shared（パラメータ追加）、cli（進化段階追加）、backend（プロンプト追加） |
| **NFR-05**: テスト | shared, cli, backend（PBT-01〜10）、cdk（スナップショットテスト） |
| **NFR-06**: 配布 | cli（クロスプラットフォームバイナリ）、web（ダウンロード案内） |
